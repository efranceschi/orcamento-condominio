=================================================
  Sistema de Orcamento - Condominio Flores
  Guia de Instalacao para Windows
=================================================

COMO USAR (Metodo Simples)
-------------------------------------------------
1. Clique duas vezes em: start_windows.bat
2. Aguarde o sistema iniciar (aparecera uma janela preta)
3. Abra seu navegador (Chrome, Edge, Firefox)
4. Digite na barra de endereco: http://localhost:8000
5. Pronto! O sistema esta funcionando

IMPORTANTE
-------------------------------------------------
- NAO feche a janela preta enquanto estiver usando
- Para fechar o sistema, feche a janela preta
- Os dados sao salvos automaticamente no arquivo:
  data/condominio_orcamento.db

REQUISITOS
-------------------------------------------------
- Windows 10 ou superior
- 4GB de RAM (minimo)
- 500MB de espaco em disco
- Navegador web moderno

SOLUCAO DE PROBLEMAS
-------------------------------------------------

Problema: "Nao foi possivel iniciar"
Solucao: Execute como Administrador
  1. Clique com botao direito em start_windows.bat
  2. Escolha "Executar como administrador"

Problema: "Porta 8000 ja esta em uso"
Solucao: Feche outros programas que usam esta porta
  Ou reinicie o computador

Problema: Navegador nao abre a pagina
Solucao: Verifique se digitou corretamente:
  http://localhost:8000
  (copie e cole no navegador)

BACKUP DOS DADOS
-------------------------------------------------
Para fazer backup dos seus orcamentos:
1. Copie o arquivo: data/condominio_orcamento.db
2. Guarde em local seguro (pen drive, nuvem, etc)

Para restaurar um backup:
1. Substitua o arquivo data/condominio_orcamento.db
   pelo arquivo de backup
2. Reinicie o sistema

ATUALIZACOES
-------------------------------------------------
Para atualizar o sistema:
1. Faca backup dos dados (veja acima)
2. Substitua todos os arquivos pela nova versao
3. Restaure o arquivo de dados
